ScienceDirect & Zotero

Guide pour Windows

Suivez ce guide pas à pas pour maîtriser l'export vers Zotero.

Démonstration Vidéo

[Vidéo de démonstration à insérer ici]
Étape 1

Connexion via la BU

Pour accéder gratuitement aux articles complets, passez toujours par le portail de votre Bibliothèque Universitaire. Authentifiez-vous avec vos identifiants étudiants.

Étape 2

Recherche sur ScienceDirect

Une fois sur ScienceDirect, utilisez la barre de recherche principale. Entrez vos mots-clés (en anglais de préférence) pour trouver des articles pertinents.

Étape 3

Sélection des Résultats

Dans la liste des résultats, identifiez les sources utiles. Cochez la case située à gauche de chaque titre pour les ajouter à votre sélection d'export.

Étape 4

Menu d'Exportation

En haut de la liste de résultats, repérez le lien "Export". Cliquez dessus pour dérouler les options d'exportation disponibles.

Étape 5

Choix du Format RIS

Choisissez l'option "Export citation to RIS".

Note importante :

Le nom "Zotero" n'apparaît pas dans le menu. C'est normal ! Le format RIS est le standard qu'il faut utiliser.

Étape 6

Téléchargement du Fichier

Le fichier sciencedirect.ris se télécharge automatiquement. Vérifiez qu'il est bien présent dans votre dossier "Téléchargements".

Étape 7

Import dans Zotero

Ouvrez Zotero et cliquez sur Fichier > Importer. Sélectionnez le fichier RIS. Une fenêtre vous confirmera le nombre de références importées.

Étape 8

Vérification et Utilisation

Vérifiez que les données (Auteurs, Dates) sont correctes dans Zotero. Vous pouvez maintenant utiliser ces références dans Word via le plugin Zotero.
